# Bioinformatics Skills

Public Codex skills for reusable bioinformatics analysis workflows.

Each top-level subdirectory is a standalone skill with its own `SKILL.md`, `agents/`, and optional `references/` or other resources. Install or copy the specific skill folder you want to use.

## Available Skills

- `microbiome-16s-analysis`: 16S amplicon and microbiome workflows covering QIIME 2, DADA2-style upstream processing, ASV/OTU cleanup, taxonomy, diversity, composition plots, LEfSe-style biomarker analysis, differential abundance, and microbiome figures.

## Planned Structure

Future skills can be added as sibling directories, for example:

- `bulk-rnaseq-analysis`
- `single-cell-rna-analysis`
- `spatial-transcriptomics-analysis`
- `omics-visualization`

Private project scripts, raw data, metadata with sensitive identifiers, local paths, and unpublished analysis notebooks should stay outside this public repository.
